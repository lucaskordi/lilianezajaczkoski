# Site Liliane Zajaczkoski - Advocacia

Site one-page profissional para a advogada Liliane Zajaczkoski.

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
